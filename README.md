💱 Conversor de Moedas

Aplicação de terminal desenvolvida em Java puro para conversão entre diferentes moedas, utilizando o Real (BRL) como moeda intermediária nos cálculos.

## 🚀 Funcionalidades

- Conversão entre 5 moedas diferentes
- Menu interativo no terminal
- Possibilidade de realizar múltiplas conversões em sequência

💰 Moedas disponíveis

| Moeda | Sigla |
|-------|-------|
| Real | BRL |
| Dólar | USD |
| Euro | EUR |
| Libra | GBP |
| Peso Argentino | ARS |

🛠️ Tecnologias

- Java 25
- Nenhuma dependência externa
